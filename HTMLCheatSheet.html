Common HTML Elements

Metadata Elements ∙ Text Elements ∙ Media Elements ∙ List Elements ∙ Other Elements ∙ Attributes

Metadata Elements

Tag Name Description
<head> Head Element Contains general information about an HTML page that isn't displayed on the page itself.
<title> Title Element Contains a text that defines the title of an HTML document.

Text Elements

Tag Name Description
<a> Anchor Element Used to create hyperlinks in an HTML document.
<h1> - <h6> Heading Elements Six different levels of headings ordered from the highest to lowest level.
<p> Paragraph Element Contains and displays a block of text.
<em> Emphasis Element Emphasizes text, typically by italicizing.
<br> Line Break Element Creates a line break in text and is especially useful where a division of text is required.

Media Elements

Tag Name Description
<video> Video Element Embeds a media player for video playback.
<img> Image Element Embeds images in documents.

List Elements

Tag Name Description
<li> List Item Element Creates list items.
<ol> Ordered List Element Creates a list of items in sequential order.
<ul> Unordered List Element Creates a list of items in no particular order.

Other Elements

Tag Name Description
<html> HTML Element The root of an HTML document containing all its content and structure.
<div> Div Element Divides an HTML document into sections.

Attributes

Tag Name Description
<target> Target Attribute Used on an <a> anchor element to specify where a hyperlink should be opened.
<alt> Alt Attribute Used to display alternative text for an <img> image element.
